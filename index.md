## Music Streaming app

### Features

- Automatic get next songs by "related videos"
- Controls -> Play, Pause, Reload, Skip and Volume.
- Download
- Media Session API

### Setup

```markdown
- Clone project

**1. Server**
- `$~ cd songr.ly-master && cd server`
- `$~ yarn` or `$~ npm install`
- `$~ yarn dev`

**2. Client**
- `$~ cd songr.ly-master`
- `$~ yarn` or `$~ npm install`
- `$~ yarn start`
```
